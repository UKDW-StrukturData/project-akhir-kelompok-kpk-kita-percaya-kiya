## Description

This repository contains the final project for the group "KPK Kita Percaya Kiya" for the Data Structure class. The project scrapes and processes data from _Mangaread.org_. It is designed as part of the coursework, tackling real-life challenges of data collection and manipulation.


## Main Features

- Web scraping from 'https://www.mangaread.org/'
- Filtering and searching---notice_tags analyzed at Entry_Form parser
