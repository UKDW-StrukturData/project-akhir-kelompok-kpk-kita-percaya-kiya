## Description

This repository contains the final project for the group "KPK Kita Percaya Kiya" for the Data Structure class. The project scrapes and processes data from _Mangaread.org_. It is designed as part of the learning process for using and implementing data scraping and AI applications.

## Main Features

- Web scraping from 'https://www.mangaread.org/'
- Data processing for further analysis
- Filtering and searching
- User management
- AI implementation for enhancing the user experience in searching for comics
- History tracking
- Comic downloading functionality
- Bookmarks
- Previous and Next chapter navigation
- Hopping between chapters
