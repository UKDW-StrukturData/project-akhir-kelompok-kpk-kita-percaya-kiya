# project-akhir-kelompok-kpk-kita-percaya-kiya

## Description

This repository contains the final project for the group "KPK Kita Percaya Kiya" at for Data Structure class. The project scrapes and processes data from _Mangaread.org_. It is designed as part of the last semester requirements.

## Main Features

- Web scraping from 'https://www.mangaread.org/'
- Data processing for further analysis
- Filtering and searching
- User management
- AI implementation for increasing user experience in searching for comics
- History tracking
- Downloading comics
- Bookmarks
- Previous and Next
- Hopping between chapters

## Technologies Used

- Python
- Streamlit
- BeautifulSoup/Scrapy (for scraping)
- Pandas (for data processing)
- Gemini API
- Matplotlib for data visualization
- Sqlite3

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/UKDW-StrukturData/project-akhir-kelompok-kpk-kita-percaya-kiya.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
